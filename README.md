# blog-site (PUBLIC)

Public Jekyll site deployed to GitHub Pages. Posts are written here **only by
the `publish.yml` workflow** (after Slack approval) — humans/automation stage
to a `drafts/` branch first, never directly to `_posts/`.

**No secrets, no automation code here, ever.** All logic lives in the private
`blog-automation` repo.

Jekyll + chirpy theme scaffold is added in **P1**. Categories: `부동산`, `일상`.
An interactive DuckDB-WASM stats dashboard is added at `/dashboard` in **P6**.
